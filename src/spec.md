# Specification

## Summary
**Goal:** Fix blog post display issues, add inline image upload to the blog editor, and restore the green color theme.

**Planned changes:**
- Debug and fix blog post listing to ensure all uploaded posts appear correctly
- Add image upload button to ReactQuill editor toolbar for inserting images within blog content
- Change website color scheme from blue-green to green throughout all components

**User-visible outcome:** Users will see all their published blog posts displayed correctly, be able to upload and insert images anywhere in their blog content while editing, and experience the website with a green color theme instead of blue-green.
