# Specification

## Summary
**Goal:** Apply a blue and green color scheme throughout the entire AyurGlow Secrets website.

**Planned changes:**
- Update CSS color palette in frontend/src/index.css to use harmonious blue and green tones, replacing the current vibrant color system (electric-blue, coral-pink, lime-green, sunshine-yellow, vivid-orange)
- Apply the new blue and green colors consistently across all components including navigation, cards, buttons, links, hero sections, and interactive elements
- Ensure both light and dark mode variants use the blue-green theme
- Maintain accessibility standards for color contrast ratios

**User-visible outcome:** The entire website displays a cohesive blue and green color scheme across all pages, components, and interactive elements while maintaining visual accessibility.
