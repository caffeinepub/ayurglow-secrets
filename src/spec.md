# Specification

## Summary
**Goal:** Add comment functionality to blog posts and enable image uploads for blog post creation/editing.

**Planned changes:**
- Add comment section at the bottom of blog post detail pages where users can view and submit comments
- Add image upload capability to blog post creation and editing forms
- Update backend to store comments with author name, text, and timestamp
- Update backend to handle image file uploads and storage

**User-visible outcome:** Users can read and post comments on blog posts, and blog authors can upload custom images when creating or editing posts instead of using image descriptions.
